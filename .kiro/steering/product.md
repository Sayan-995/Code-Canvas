# Code Canvas Clone

A React + TypeScript application that visualizes code files as floating nodes on an infinite canvas.

## Core Features

- **Folder Upload**: Select local folders to visualize their file structure
- **GitHub Import**: Import public GitHub repositories via Octokit
- **Infinite Canvas**: Navigate files using React Flow-powered canvas with pan/zoom
- **File Preview**: View and edit file contents directly on canvas nodes
- **Static Analysis**: Real-time ESLint and TypeScript diagnostics with 500ms target
- **Collaboration**: Real-time file sync via Socket.IO backend
- **Code Explanation**: AI-powered code explanations via Gemini integration
- **Repository Segmentation**: Split large repos into manageable segments

## Target Users

Developers who want to explore and understand codebases visually, with support for collaborative review sessions.
